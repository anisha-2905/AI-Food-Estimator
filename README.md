# NutriVision AI – Food Calorie Estimator

NutriVision AI is a student project that uses Artificial Intelligence and Computer Vision to recognize food from uploaded images and estimate nutritional information, including calories, protein, carbohydrates, fats, and fiber. The project demonstrates the integration of AI-powered image recognition with a modern and user-friendly web application.

---

## Project Overview

NutriVision AI allows users to upload a food image and receive an AI-generated prediction of the food item along with its estimated nutritional values. The application also provides a dashboard with meal history, nutrition insights, and interactive visualizations to help users understand their eating patterns.

This project showcases the integration of AI-powered image recognition with a modern web application while maintaining a clean and user-friendly interface.

---

## Features

- Upload food images
- AI-powered food recognition
- Estimated calories and nutritional values
- Confidence score for predictions
- Macronutrient breakdown
- Meal history tracking
- Nutrition insights dashboard
- Weekly calorie visualization
- Search and filter meal history
- Fully responsive design
- Modern dashboard interface

## 🛠️ Installation

1. Install dependencies:
  ```bash
  npm install
  # or
  yarn install
  ```

2. Start the development server:
  ```bash
  npm run dev
  # or
  yarn dev
  ```
3. Open [http://localhost:4028](http://localhost:4028) with your browser to see the result.
